Org-Mode

Org-Mode是由Carsten Dominik于2003年创建的，最初是为了组织他的生活和工作，自从第一次发布以来，许多用户和开发人员已经为这个免费软件包做出了贡献，作为Emacs工作的主要模式，Bastien Guerry是目前的维护者，并与快速发展的社区合作。自从Emacs成功以来，其他一些系统也开始提供使用org文件的功能。

Quick Start

Introduction

Org-Mode是文本编辑软件Emacs的一种支持内容分级显示的编辑模式，其核心是一个简单的大纲，用于笔记和列表管理，概述，笔记，超链接，电子表格，TODO列表，项目规划，GTD（Getting Things Done），HTML和LaTeX创作，都可以在Emacs中使用纯文本文件。比如TODO列表可以给每一件事分别赋上不同的优先级和最后完成期限，每一个TODO项目又可以细分成次级TODO列表或者核对表，而且每一个项目还都可以分别加标签或者属性。除此之外，它还可以自动生成某一段时间以内的日程表。

Set Up

Org-Mode有如下几个配置：Inbox，来源于GTD，快速输入临时事件或突然的想法，释放头脑内存；Tasks，相当于GTD中的TODO-list，即那些一步就能完成的零散事件；Projects，需要多个步骤完成的项目，每个项目由二级标题三级标题组成，也可再细分；Routines，一些每天都要进行的习惯，使用Org-mode可以很好地进行重复；Remind，需要提醒的事情，比如某个时间点上课；Reading，正在看的书；Weekly Review，每周要结束时进行回顾，其中有一些代码段，用于进行当周的时间统计等。
